# Agile Wheel - Interactive Project Management Platform

A modern, collaborative project management tool designed specifically for software developers and scrum masters. Track your agile development lifecycle through an intuitive, interactive wheel interface.

## What Sets Us Apart

- **Visual Agile Workflow**: Navigate through agile stages with an interactive wheel interface
- **Real-time Collaboration**: Seamless coordination between developers and project managers
- **Zero Framework Overhead**: Built with pure HTML, CSS, and JavaScript for maximum performance
- **Intuitive Design**: Beautiful, modern UI that makes project tracking effortless
- **Developer-First**: Built by developers, for developers

## Tech Stack

- Pure HTML5
- Modern CSS3 with animations
- Vanilla JavaScript (ES6+)
- Proxy-based reactive state management

## Getting Started

1. Clone this repository
2. Open `index.html` in your browser
3. Create an account or login
4. Start managing your agile projects!
